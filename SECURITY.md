# Security

Please do not report security issues in public GitHub issues.

If you believe you found a vulnerability or accidentally exposed private data,
contact the JoomPulse team through the official JoomPulse contact channel:

https://joompulse.com

When reporting a problem, include:

- A short description of the issue.
- The affected file or workflow.
- Steps to reproduce, if applicable.
- Whether any secret, private URL, or private data may have been exposed.
